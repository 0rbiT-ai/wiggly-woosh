# WigglyWoosh — Professional Pet Care Platform

WigglyWoosh is a comprehensive, production-ready pet care application designed for modern pet parents. It provides a secure medical record vault, a verified marketplace for premium products, a supportive community forum, and an integrated veterinary booking system.

## 🚀 Features

- **Health Vault**: Securely manage pet vaccinations and medical records with local persistence.
- **Smart Symptom Checker**: AI-powered assistant UI for preliminary pet health checks.
- **Verified Marketplace**: Shop for veterinarian-recommended products with a functional cart system.
- **Community Forum**: Discuss pet care with other parents and get advice from verified experts.
- **Vet Search & Booking**: Find clinics on an interactive map and schedule appointments directly.

## 🛠 Tech Stack

- **Framework**: React 19 (TypeScript)
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Persistence**: LocalStorage API

## 📦 Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Production Build
```bash
npm run build
```

## 🔒 Security & Privacy
WigglyWoosh prioritizes pet data privacy. Currently, all data is stored locally on the user's device using `localStorage`. Future updates will include end-to-end encrypted cloud synchronization.
